# DEAPC-P14---GEST-O-DE-PROJECTOS


Objetivos da aplicação:

Os sistemas de gestão de projetos requerem uma aplicação web com uma interação com um servidor de base de dados 
para gerir os dados do projeto, informação de equipas, e listas de tarefas. A aplicação web precisa de interagir com o 
servidor de base de dados para gerir os dados do projeto, acompanhar a atividade da equipa, e atualizar listas de tarefas.



Funcionalidades do projeto para os diferentes utilizadores:

Cliente ( Técnico da empresa...)

-Visualização do Calendário: Consultar prazos de entrega e reuniões de projeto de forma mensal/semanal.

-Detalhes do Projeto: Clicar num projeto no calendário para ver a descrição, documentos anexados e equipa responsável.


Administrador (Gestor de projetos)

- Criar projetos: Adicionar projetos novos no calendário

- Atribuição de Tarefas: Arrastar projetos para mudar datas (funcionalidade avançada) ou editar quem está alocado a cada data.
